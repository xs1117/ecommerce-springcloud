# AI客服知识库

退货申请流程：
1. 用户提供订单号。
2. AI客服识别到退货意图后，先生成待确认操作，不直接执行。
3. 用户确认后，AI客服调用售后动作接口发起 `APPLY_RETURN`。
4. 若用户取消，则清除待确认操作，不执行任何业务动作。

常见提示语：
- 如果用户未提供订单号，提示其补充订单号。
- 如果涉及需要写操作（退货、换货、退款），必须明确告知将进行二次确认。
- 对于一般咨询类问题，优先引用平台规则并给出简洁回答。

异常处理：
- 登录态无效时，提示用户重新登录。
- 售后会话创建失败时，提示稍后重试。
- 接口超时或下游失败时，不要重复提交操作，建议用户稍后再试。
